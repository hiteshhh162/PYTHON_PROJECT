# 🎮 Hangman Game in Python

A simple and fun **Hangman Word Guessing Game** built using Python 🐍

This is a beginner-friendly terminal-based project where the player has to guess the hidden fruit name letter by letter before running out of chances.

---

# 🚀 Features

✅ Random word selection  
✅ Limited chances system  
✅ Letter-by-letter guessing  
✅ Win/Lose conditions  
✅ Beginner-friendly logic  
✅ Clean terminal output  

---

# 🧠 Concepts Used

This project helped practice:

- Python Variables
- Loops (`while`)
- Conditional Statements (`if-else`)
- Lists
- Strings
- Random Module
- User Input
- String Joining (`join`)
- Logic Building
