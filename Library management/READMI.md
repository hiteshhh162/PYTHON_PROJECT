# 📚 Library Management System

A simple and powerful **Terminal-Based Library Management System** built using **Python**.
This project helps manage books, members, borrowing, and returning books using a JSON database.

---

# 🚀 Features

✅ Add Books
✅ View Books
✅ Add Members
✅ View Members
✅ Borrow Books
✅ Return Books
✅ JSON File Storage
✅ Random Book & Member IDs
✅ Error Handling
✅ Object-Oriented Programming (OOP)

---

# 🛠️ Technologies Used

* Python
* JSON
* OOP Concepts
* File Handling
