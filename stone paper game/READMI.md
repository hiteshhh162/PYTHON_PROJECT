# 🎮 Stone Paper Scissor Game in Python

A fun and interactive **Stone Paper Scissor** game built using Python.
In this game, the player competes against the computer, and the first to score **5 points** wins the match. 🏆

---

# 🚀 Features

* 🎯 User vs Computer gameplay
* 🏆 Live score tracking
* ⚠️ Invalid input handling
* 🎲 Random computer moves
* ✨ Attractive terminal UI with emojis
* 🥳 Winner announcement system

---

# 🛠️ Technologies Used

* Python 🐍
* `random` module
* `time` module

# 🎮 Game Rules

* 🪨 Stone beats ✂️ Scissor
* 📄 Paper beats 🪨 Stone
* ✂️ Scissor beats 📄 Paper
* 🤝 Same choices result in a draw
* 🏆 First player to score 5 points wins the game
