# 🎮 Stone Paper Scissor Game (Python)

A simple and fun **Stone Paper Scissor** game built using **Python**.
Play against the computer and try to reach **5 points first** to win the game! 🏆

---

## 🚀 Features

* Interactive command-line gameplay
* User vs Computer mode
* Score tracking system
* Winner announcement
* Input validation
* Emoji-based attractive UI

---

## 🛠️ Technologies Used

* Python 🐍
* Random Module
* Time Module

## ▶️ How to Run

1. Install Python on your computer
2. Clone this repository

```bash
git clone https://github.com/your-username/stone-paper-scissor.git
```

3. Open the project folder

```bash
cd stone-paper-scissor
```

4. Run the program

```bash
python main.py
```

---

## 🎯 Game Rules

* 🪨 Stone beats ✂️ Scissor
* 📄 Paper beats 🪨 Stone
* ✂️ Scissor beats 📄 Paper
* First player to score **5 points** wins the game

---

## 📸 Sample Gameplay

```bash
🏆 Your Score : 2 | 🤖 Computer Score : 1

1 ➜ 🪨 Stone
2 ➜ 📄 Paper
3 ➜ ✂️ Scissor

👉 Enter your choice: 1
```
