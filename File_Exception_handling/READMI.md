# 🌟 File & Folder Manager using Python 🌟

A powerful and beginner-friendly **File & Folder Manager** built using Python.  
This project helps users perform complete CRUD operations on files and folders directly from the terminal.

> 📂 Create • 📖 Read • ✏️ Update • ❌ Delete

---

# 🚀 Features

## 📁 Folder Management
✅ Create Folder  
✅ View All Folders  
✅ Rename Folder  
✅ Delete Folder (Even if files exist inside)

---

## 📄 File Management
✅ Create File  
✅ Read File  
✅ Rename File  
✅ Append Content  
✅ Overwrite Content  
✅ Delete File

---

# 🛠 Technologies Used

| Technology | Purpose |
|---|---|
| Python | Core Programming |
| pathlib | Path Handling |
| shutil | Folder Deletion |
| File Handling | File Operations |
| Exception Handling | Error Management |
